# Asociación de Residentes San Patricio Meadows

Sitio público estático para la Asociación de Residentes San Patricio Meadows.

## Vista Local

Abra `index.html` directamente en un navegador o sirva esta carpeta con cualquier servidor estático.

## Publicación

Este sitio está preparado para GitHub Pages con el dominio personalizado:

```text
sanpatriciomeadows.com
```

El DNS en GoDaddy debe apuntar el dominio principal a GitHub Pages y `www` a `pabloiv.github.io`.
